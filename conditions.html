Les STRUCTURES CONDITIONNELLES

Lorsque l'on veut que du code soit exécuté si une certaine condition est vrai, on va utilise une structure IF.

Syntaxe :


if (condition) {
    bloc d'instructions...
}

SINON

On peut définir le code qui sera exécuté si la condition est fausse.

Syntaxe : 


if(condition) {
    bloc d'instructions...
} else {
    bloc d'instructions...
}

OPÉRATEURS DE COMPARAISON

  • inférieur : <
  • inférieur ou égal : <=
  • supérieur : >
  • supérieur ou égal : >=
  • égal à : ==
  • strictement égal : ===
  • différent de : !=
  • strictement différent de : !==

BOOLEEN

Dans les parenthèses du if, il faut une expression booléenne. Quelque soit l'expression, javascript va la convertir en booléen. Selon le type de l'expression, une valeur est considérée comme false, les autres sont considérée comme true.

Type Valeur false
string chaîne de caractère vide ""
number 0, NaN
boolean false
null null
undefined undefined

OPÉRATEURS LOGIQUES

Les opérateurs logiques s'utilisent avec des expressions booléennes et retourne un résultat booléen. Ils sont utilisés quand on a une condition constituée de plusieurs conditions.

Opérateur Signification Explication
&& AND (ET) Les 2 conditions doivent être vraies en même temps
|| OR (OU) Au moins 1 des 2 conditions doit être vraie
! NOT (NON) Inverse une valeur booléenne

L'opérateur NOT ! est un opérateur unaire (c'est-à-dire qu'il ne s'applique que sur 1 seule expression, contrairement aux opérateurs binaires qui ont besoins de 2 expressions).

STRUCTURE SWITCH


switch(expression) {
    case valeur1:
        bloc d'instructions 1...
    break;

    case valeur2:
        bloc d'instructions 2...
    break;
    ...

    default:
        bloc d'instructions
}

OPERATEUR TERNAIRE ou IF INLINE

On peut utiliser l'opérateur ternaire lorsque la valeur d'une expression dépend d'une condition.

Syntaxe : 


condition ? valeur_si_condition_vraie : valeur_si_condition_fausse
